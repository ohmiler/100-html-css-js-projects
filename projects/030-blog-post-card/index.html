เริ่มต้น CSS Grid สำหรับมือใหม่

CSS Grid ช่วยจัดเลย์เอาต์หลายคอลัมน์ได้ง่ายมาก แค่กำหนด display: grid และ grid-template-columns ก็สร้างการ์ดเรียงกันได้ทันที บทความนี้จะพาเรียนรู้จากพื้นฐานจนทำ component จริงได้ รวมถึง responsive บนมือถือด้วย media query เพื่อให้ layout ยืดหยุ่นตามขนาดหน้าจอ
