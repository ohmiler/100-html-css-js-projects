## ตัวอย่างเต็มรูปแบบ: EP01 ปุ่มเปลี่ยนสีเมื่อคลิก

**video_title:** `[EP01] ทำปุ่มเปลี่ยนสีเมื่อคลิก ด้วย HTML CSS JS | สอนมือใหม่`

**estimated_minutes:** 18

**concepts_new:** `addEventListener`, `classList.toggle`

**concepts_review:** (ไม่มี — ตอนแรกของคอร์ส)

---

### 0:00–0:30 — Hook

> "สวัสดีครับ ยินดีต้อนรับสู่ EP01 ของคอร์ส 100 โปรเจกต์ HTML CSS JS สำหรับมือใหม่ วันนี้เราจะทำปุ่มที่กดแล้วเปลี่ยนสีแบบนี้ [คลิกสาธิต] ตอนจบคุณจะเข้าใจพื้นฐาน HTML structure, CSS class, และ JavaScript event listener มาเริ่มกันเลย"

**สิ่งที่แสดงบนจอ:** ผลลัพธ์สุดท้าย — ปุ่มสีน้ำเงิน คลิกแล้วเปลี่ยนเป็นสีแดง คลิกอีกทีกลับ

---

### 0:30–5:00 — HTML

**สิ่งที่ทำบนจอ:**

1. สร้าง `index.html` พื้นฐาน (`<!DOCTYPE html>`, `lang="th"`, meta charset)
2. เพิ่ม `<button id="toggleBtn" class="btn">กดเปลี่ยนสี</button>`
3. link `style.css` และ `script.js`
4. เปิด Live Server ดูผลลัพธ์ (ปุ่มยังไม่สวย)

**สิ่งที่พูด (สั้นๆ):**

> "สร้างไฟล์ index.html ก่อน ใส่ปุ่มหนึ่งปุ่ม ให้ id กับ class ไว้เพื่อเอาไป style และเขียน JS ทีหลัง"

**ยังไม่พูดเรื่อง:** semantic HTML ลึก, accessibility ละเอียด

---

### 5:00–12:00 — CSS

**สิ่งที่ทำบนจอ:**

1. Reset พื้นฐาน (`box-sizing`, `body` center ด้วย flexbox)
2. Style `.btn` — padding, border-radius, cursor, สีน้ำเงิน, transition
3. สร้าง class `.btn--active` สีแดง
4. ทดสอบเพิ่ม class ใน DevTools ดูผลลัพธ์

**สิ่งที่พูด (สั้นๆ):**

> "ใส่ CSS ให้ปุ่มสวย สร้าง class สำหรับสถานะ active แยกไว้ จะได้สลับด้วย JavaScript ทีหลัง"

---

### 12:00–17:00 — JavaScript

**สิ่งที่ทำบนจอ:**

1. สร้าง `script.js`
2. `const btn = document.getElementById('toggleBtn')`
3. `btn.addEventListener('click', () => { btn.classList.toggle('btn--active') })`
4. ทดสอบคลิก — สีเปลี่ยนได้

**สิ่งที่พูด (สั้นๆ):**

> "เลือกปุ่มจาก DOM ด้วย getElementById แล้วฟัง event click เมื่อกดให้ toggle class active"

---

### 17:00–19:00 — อธิบายหลักการ

**Concept 1: DOM และ getElementById**

> "DOM คือต้นไม้ของ element ในหน้าเว็บ JavaScript เข้าถึง element ผ่าน id, class, หรือ querySelector"

**Concept 2: addEventListener**

> "addEventListener บอกว่าเมื่อเกิด event อะไร (เช่น click) ให้รัน function นี้ ดีกว่า onclick ใน HTML เพราะแยก logic ออกจาก structure"

**Concept 3: classList.toggle**

> "classList จัดการ class ของ element toggle คือมีอยู่แล้วเอาออก ไม่มีก็ใส่เข้า — เหมาะกับสถานะเปิด/ปิด"

---

### 19:00–20:00 — ท้าทาย

**challenge_th:** ลองเปลี่ยนเป็นสีเขียวแทนสีแดง หรือเพิ่มข้อความบนปุ่มว่า "เปิด" / "ปิด" เมื่อสลับสถานะ

> "ท้าทายหลังจบตอน: ปรับสี active เป็นสีที่คุณชอบ หรือเปลี่ยนข้อความบนปุ่มตามสถานะ ลองทำดูแล้วเจอกัน EP02 ครับ"
