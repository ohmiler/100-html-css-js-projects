# 100 HTML/CSS/JS Projects — แผนหลักสูตร

คอร์สวิดีโอภาษาไทยสำหรับมือใหม่ที่คัดลอกโค้ดได้ แต่อยากเข้าใจหลักการและฝึกทำเอง โดยพาทำ component เล็ก 100 ชิ้นด้วย Vanilla HTML, CSS, JavaScript

---

## กลุ่มผู้เรียนเป้าหมาย

- เคยเห็นหรือคัดลอกโค้ด HTML/CSS/JS มาบ้าง
- อยากเข้าใจว่าโค้ดทำงานอย่างไร ไม่ใช่แค่ copy-paste
- ไม่จำเป็นต้องมีประสบการณ์เขียนโปรแกรมมาก่อน

---

## สิ่งที่จะได้จากคอร์สนี้

- ทำโปรเจกต์ UI component ได้ 100 ชิ้น
- เข้าใจ DOM, event, layout (flex/grid), form, animation
- มีพอร์ตโฟลิโอโค้ดจริงใน GitHub
- พื้นฐานพร้อมเรียน framework (React, Vue) ในอนาคต

## รูปแบบการสอน

**Build-first:** พาทำก่อน → อธิบายทีหลัง

1. โชว์ผลลัพธ์สุดท้าย
2. สร้าง HTML
3. ใส่ CSS
4. เพิ่ม JavaScript
5. อธิบายหลักการ 2–3 concept
6. ท้าทายให้ปรับแต่งเอง

เทมเพลตละเอียด: [video-template.md](video-template.md)

## เครื่องมือที่ต้องมี

- เบราว์เซอร์ (Chrome หรือ Firefox)
- [VS Code](https://code.visualstudio.com/)
- Extension: **Live Server**
- ไม่ต้องใช้ Node.js, npm, หรือ framework
